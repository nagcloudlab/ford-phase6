Meet Our Service: UPI Transfer

A real-world Spring Boot app we'll use throughout Phase 6

A simplified UPI money transfer service built with Spring Boot 3.4. Think of it as a mini Google Pay backend — accounts, balances, and instant transfers.

1
Transfer Money

POST /api/transfer — send money between UPI IDs

2
Check Balance

GET /api/balance/{upiId} — real-time account balance

3
Transaction History

GET /api/transactions/{upiId} — full audit trail

4
Health & Docs

Actuator health checks + Swagger UI out of the box

Tech Stack
Spring Boot  3.4.4
Java         17
Database     H2 (in-memory)
ORM          Spring Data JPA
Validation   Jakarta Bean
API Docs     SpringDoc OpenAPI
Monitoring   Spring Actuator
Why this service?
Small enough to understand in 5 minutes, rich enough to demonstrate every cloud concept we'll cover.
